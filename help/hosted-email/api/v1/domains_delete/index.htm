Delete domain¶

Delete domain (verified or pending verification)

Caution

Use with caution as this would delete everything associated with a verified domain like accounts, aliases, sender addresses & domain aliases

  • HTTP Method: DELETE
  • URL: https://example.domain.tld/api/v1/domains/
  • Require authentication: Yes
  • Permission level required: MasterAdmin / DomainAdmin

Request parameters¶

Parameter Type Description Required
domain str Domain name Required

Response¶

HTTP 204 NO CONTENT (empty string)

Example request (Python)¶

import requests
import json
from pprint import pprint

url = "https://example.domain.tld/api/v1/domains/"
data = {
"domain" : "test1.vonmail.com"
}
r = requests.delete(url, json=data, auth=("admin","long-auth_token-here"))
try:
    pprint(r.json())
except:
    print(r.text)
print(r.status_code)
